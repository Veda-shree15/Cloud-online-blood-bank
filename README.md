# Cloud-Based Blood Bank Management System

A cloud-based Blood Bank Management System developed to manage blood donation, donor registration, blood requests, and appointment scheduling efficiently. The system connects donors, recipients, and blood banks through a centralized online platform.
It allows users to register as donors, request blood during emergencies, check blood availability, and schedule donation appointments. Admins can manage donor details, blood stock, and emergency requests through an organized dashboard.
Firebase is used for cloud database management, user authentication, and real-time data storage, making the system fast, secure, and easily accessible from anywhere.

# Features
* Donor Registration and Login
* Blood Donation Scheduling
* Blood Request Management
* Blood Group Availability Tracking
* Admin Dashboard
* Emergency Blood Request Handling
* Secure Cloud Storage using Firebase
* Real-Time Database Updates

* Frontend: HTML, CSS, Bootstrap, JavaScript
* Backend: PHP
* Database: Firebase Realtime Database
* Authentication: Firebase Authentication

# Objective
The main objective of this project is to develop a cloud-based platform for efficient blood bank management using Firebase cloud services. It helps store donor and recipient data securely, manage blood requests in real time, improve emergency response, and reduce manual work through a centralized online system.
